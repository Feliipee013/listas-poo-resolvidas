# 📘 Repositório de Programação Orientada a Objetos (POO)

Este repositório contém resoluções das listas de exercícios da disciplina de **Programação Orientada a Objetos (POO)**.

Além de termos cursado e concluído a disciplina, atualmente atuamos como monitores, e este material foi criado com o objetivo de ajudar outros estudantes no aprendizado dos conceitos fundamentais.

---

## 🎯 Objetivo

- Disponibilizar soluções claras e organizadas das listas  
- Servir como material de apoio para estudo  
- Demonstrar boas práticas de programação  
- Auxiliar alunos que estão cursando POO  

---

## 🧠 Conteúdos abordados

As listas resolvidas neste repositório incluem temas como:

- Classes e Objetos  
- Encapsulamento  
- Herança  
- Polimorfismo  
- Métodos e Construtores  
- Estruturas condicionais e de repetição  
- Manipulação de dados  

---

## 📂 Estrutura do Repositório

```
📁 src(listas)/
   ├── lista1/
   ├── lista2/
   ├── lista3/
   └── ...
```

Cada pasta contém os exercícios resolvidos de forma organizada, podendo incluir:

- Código-fonte
- Comentários explicativos
- Testes simples

---
